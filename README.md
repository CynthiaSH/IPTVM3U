# IPTV LINK Share
M3U直播地址、IPTV直播源分享



## 测试环境描述
> 连通性测试工具：IPTV Checker 
> 
> 测试线路：China Unicom
> 
> 通过标准：连通性成功即可，部分信号可能经过转播，非官方或大型运营商提供，速度会慢



> 播放工具：
> 
> * Windows端：Potplayer
> 
> * 电视端：Kodi
> 
> * 手机端：nPlayer
